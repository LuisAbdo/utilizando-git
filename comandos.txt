
-> Sincroniza sem alterar a pasta local verificando mudanças no Github
git fetch
-> Configura o pc com o nome e E-mail do responsável pelo código 
git config --global user.name "Luis Abdo"
git config --global user.email luis.abdo2006@gmail.com
->incializa um repositório git (utilizado apenas uma vez)
git init

-> verificar o status do repositório
git status
_>adicionar os novos arquivos no git para versoinar 
git add .

-> Salva uma versão dos arquivos 
git commit -m "Porque estou salvando???"

-> envia o que está no seu computador para o github
git push 

->Baixa a atualização dos arquivos do Github para uma pasta já existente
git pull
